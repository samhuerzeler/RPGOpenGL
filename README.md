========================
RPGOpenGL
========================

Keybindings
------------------------
ESC		Quit Game

F1              Disable lighting

F2              Enable lighting

F5              Switch polygon mode

W		Move forward

S		Move backward

A		Strafe left

D		Strafe right

Q		Turn left

E		Turn right

F		Start auto attack

1		Use ability 1

2		Use ability 2

num_1           Equip sword

num_2           Equip bow
