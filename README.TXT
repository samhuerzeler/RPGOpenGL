========================
RPGOpenGL
========================

1. Doubleclick on RPGOpenGL.jar OR use the commandline: java -jar RPGOpenGL.jar
2. Enjoy!

Keybindings
------------------------
ESC		Quit Game
W		Move forward
S		Move backward
A		Turn left
D		Turn right
Q		Strafe left
E		Strafe right
F		Attack
1		Change to sword if looted
2		Change to Bow if looted

Loot
------------------------
Goblin	Bow
Orc		Sword, Helm (not usable yet ;-))

Monsters
------------------------
Orc		Red spawn point (middle)
Goblins	Red spawn points (left, right)
Guard	Blue spawn point
Player	Green spawn point
